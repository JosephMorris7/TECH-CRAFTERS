# Opportunity Tracker

This project is a prototype that converts scattered internship notifications 
from WhatsApp, Telegram, and Email into structured opportunities.

## Problem
Students receive internship opportunities across multiple platforms and 
often miss deadlines while searching through message threads.

## Solution
This prototype extracts:
- Company name
- Role
- Eligibility
- Deadline
- Application link

and organizes them in a dashboard.

## Note
This is a UI prototype generated with Claude using React JSX.
